# chromadb-WebUI

To management your chromadb in friendly Web UI

## Getting Started

First, install the dependency

```bash
npm install
```


Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
